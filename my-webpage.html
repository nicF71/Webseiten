Der Webentwickler Kurs

Tag-Übersicht

Head-Tag: Ganzes Wissen
Body-Tag: Infos wie zB Text
Div-Tag: Organisiert die Struktur einer Webseite
Title-Tag: Was Oben im Tab angezeigt wird
Meta-Tag: Grundsätzliche Information (charset = Welcher Zeichensatz, http-equiv = Contenttypen, content = Inhalttypen, name = viewport = definiert Breite und Zoomfaktor)
Style-Tag: Kann zB das Aussehen der Webseite bearbeiten mit anderer Sprache und auch strukturieren
H1-Tag -> Header Tag: Überschrift (h1 ist das größte, dann h2, usw. bis h6, danach kein header mehr)
P-Tag -> Paragraph Tag: Ein Paragraph Text (Kreiert Zeilenumbrüche)
Br-Tag: sorgt für Zeilenumbruch

Bilder

Bilder müssen im richtigen Ordner als .png gespeichert werden (Der Ordner, in dem auch der Code gespeichert wird) oder die Bildadresse muss richtig angegeben werden.
Bilder werden mit dem img-Tag aufgerufen, in welchem nun noch alle Informationen zu dem Bild mit dem src-Tag definiert werden müssen.
In dem img-Tag kann man auch das Bild bearbeiten (width= beeinflusst die breite und skaliert das Bild, height= beeinflusst die höhe und skaliert das Bild).
Nur wenn width und height angegeben wird wird beim zweiten parameter nicht mehr skaliert.
Die Einheit für die Parameter ist Pixel.

Der align-Befehl funktioniert auch bei Bildern

Beispiel:

Dies ist Homer Simpson

Dies ist Marge Simpson

Um einen Platzhalter in den Boxen zu definieren wird die methode placeholder im Input-Tag benutzt.
